# @meshpack/crypto

MeshPack E2EE v1 — TypeScript kripto paketi.

Klinik/lab Tauri uygulamaları ve web istemcileri bu paketi kullanır. Flutter tarafı aynı wire format için Dart implementasyonu içerir (`apps/mobile/lib/crypto/`).

```bash
cd packages/crypto
npm install
npm run check
```

Protokol: [`docs/E2EE_PROTOCOL.md`](../../docs/E2EE_PROTOCOL.md)
